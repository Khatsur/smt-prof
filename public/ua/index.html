Обладнання і матеріали для складання електроніки

SMT PROF все для SMT

Монтаж на поверхню плат

Обладнання для монтажу на поверхню SMT і в отвори плат THT. Автоматичні склади, SPI і AOI, конвеєри, ПО для виробництва.

Селективне паяння

Обладнання для селективного паяння компонентів в отвори плат. Окремо-стоячі машини і обладнання для роботи у виробничих лініях.

Селективне лакування

Селективне нанесення захисних покриттів на електронні плати. Автоматизація процесів дозування матеріалів, герметизації, заповнення.

Миття плат і трафаретів

Устаткування для миття електронних плат після пайки. Струменеві машини для миття трафаретів. Ощичищення частина обладнання для пайки та оснастки.

Малая серія SMT

Обладнання для мало-серійного виробництва і протипування. Маніпулятори та напівавтоматичні пристрої для установки SMD на плати.

Розділення плат

Устаткування для розділення групових заготовок друкованих плат. Ручні пристрої і автоматизовані машини. Магазини для плат.

Матеріали для очищення

Рідини для відмивання електронних плат, трафаретів і технологічної оснастки. Засоби контролю забруднень і процесу миття.

Паяльна паста і припій

Паяльна паста для нанесення через трафарет. Припій з флюсом для ручного паяння. Припій в брусках для машин пайки хвилею припою.

Захисні покриття

Захисні покриття для електронних плат: акрилові, силіконові, уретанові, гумові. Лаки з УФ затвердінням. Засоби для зняття лаку.
